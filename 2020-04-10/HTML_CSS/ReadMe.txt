HTML5CSS3
8강 CSS3 선택자2
1. 태그 선태자
 - 태그 선택자에 속성을 선택해서 css를 적용 한다.

2. 링크 선택자.  -거의 안쓰지만 혹시 모르기 때문에 확인만 하자
 - <a> 캐그를 선택할 때 사용 한다.

3. 부정 선택자
 - :not 선택자를 이용해서 명시한 선택자들 외의 html 요소를 선택
 - not을 지정되지 않은 태그에 css 가 적용된다.

4. 문자 선택자
 - first-letter 선택자
   : 첫번째 문자를 선택해서 CSS 를 적용
 - first-line 선택자
   : 첫번째줄을 선택해서 CSS를 사용.

5. 후손 및 자손 선택자
 - 후손 선택자
   : 특정 html태그의 아래에 있는 모드 태그 요소를 CSS 적용한다.
 - 자손 선택자
   : 특정 html 태그의 바로 아래에 있는 태그 요소를 CSS를 적용한다.

6. 동위, 반응, 상태 선택자
 - 동위 선택자
   : html태그 계층구조를 고려하여 동등한 위치에 있는 태그요소를 CSS 를 적용한다.
   ex)   동위   나가리
	<div>
		<p>
	</div>
	<div>
		<p>
	</div>
	<p>
 - 반응 선택자
   : 웹 사이트를 이용하는 사용자의 반응에 따라 html 태그요소를 선택한다.
   ex) :hover 마우스가 태그위에 올라왔을때 CSS 가 적용된다.
 - 상태 선택자
   : html태그 요소의 상태에 따라서 선택한다.
