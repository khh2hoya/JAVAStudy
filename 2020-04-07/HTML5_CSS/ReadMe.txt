5강 HTML5 태그 4

1. 페이지 레이아웃 설정 태그
 - block 타입의 태그   / <p> , <div> 태그
 - inline 타입의 태그  / <img>, <span> 태그 
 - 구분하기 위해 <div> 태그를 사용하여 layout 을 설정 한다.

2. 시멘틱 태그
 - HTML5 에서부터 사용.
 - 검색을 최적화 하기위해 사용.
 - <header> : 페이지 헤드 영역
 - <footer> : 페이지 하단 영역
 - <nav>    : 페이지 메뉴 영역
 - <section>: 페이지 내용
 - <article>: section과 유사하며 글 또는 내용 출력
 - <aside>  : 부가적인 내용.
 - <div>처럼 영역을 구분해주고 검색시 도움을 줄수 있다.
